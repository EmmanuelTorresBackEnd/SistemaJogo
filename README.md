# 🎮 SistemaJogo

## 📌 Descrição
Projeto de jogo desenvolvido em **C#/.NET**, com foco em lógica de programação, organização de classes e simulação de mecânicas básicas.  
Este sistema foi criado para explorar conceitos de **POO (Programação Orientada a Objetos)**, **estruturas de dados** e **interatividade**.

## 🛠️ Tecnologias Utilizadas
- C# 10
- .NET 6/8 SDK
- Visual Studio 2022 / VS Code
- GitHub para versionamento

## 🎯 Objetivos
- Implementar lógica de jogo com classes e objetos.
- Explorar herança, polimorfismo e encapsulamento.
- Criar mecânicas básicas (movimento, pontuação, combate, inventário).
- Simular interações entre personagens e ambiente.
